üìö How Enqueue Works

The enqueue operation adds a new node to the end (tail) of the queue. Here's the logic breakdown:

if (this.isEmpty()) { this.head = this.tail = newNode; } else { this.tail.next = newNode; this.tail = newNode; } this.sizeCount++;

üîç Step-by-Step Logic:

  1. Check if queue is empty:
    • If empty: Set both head and tail to the new node (it's the only node)
    • If not empty: Proceed to step 2
  2. Link the new node: Set the current tail's next pointer to the new node
  3. Update tail: Move the tail pointer to the new node (it's now the last element)
  4. Increment size: Increase the queue size counter
